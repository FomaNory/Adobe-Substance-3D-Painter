# Adobe-Substance-3D-Painter
Installer and patcher for Adobe Substance 3D Painter.

---

## Features  
- **Automatic download**  
  Automatically downloads the program file from Adobe servers.  
- **Smart patch application**  
  Fully automatic patching process control adapting to program updates.
